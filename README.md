<h1>英語-日本語の単語帳テスト作成Excel</h1>

ファイルは"Dictionary"，"Test"，"Answer"からなります．
Dictionaryに"follow，〜に従う"というように記載してください．

「*表示*」→ 「*マクロの表示*」→ 「*実行*」
